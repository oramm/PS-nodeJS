# Bug Fix Loop Prompt

You are running a local bug-fix loop for PS-nodeJS.

## Input

- `bug-context.json` generated by `yarn bugfix:run`
- repository path from `repo_path`
- mode: `analyze` or `analyze+patch`

## Required workflow

1. Select the highest priority bug from `bugs`.
2. Diagnose root cause using stack, message, and request context.
3. Identify exact files and symbols to change.
4. If mode is `analyze+patch`, implement the smallest safe patch.
5. Run targeted tests first, then broader relevant tests.
6. Produce a short review summary with risks and follow-ups.
7. Provide output object:

```json
{
    "bug_id": 0,
    "diagnosis": "",
    "files": [""],
    "changes": [""],
    "tests": [{ "command": "", "result": "pass|fail|not-run" }],
    "risks": [""],
    "status": "triaged|in_progress|waiting_human|fixed"
}
```

## Guardrails

- Do not auto-merge.
- Do not auto-deploy.
- Keep human checkpoint before merge.
- Prefer deterministic fixes over speculative refactors.
